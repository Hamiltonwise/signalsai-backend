# Pre-Spec Strategic Check

Run this BEFORE writing any spec, blueprint, or architecture document.
If any question can't be answered, STOP and research before proceeding.
This prevents the "one more thing" pattern that wastes bandwidth.

## 1. VIRAL MECHANICS
- [ ] Does this feature create passive distribution? (Score badge, shared links, embedded widgets)
- [ ] Does this feature create active sharing triggers? (Competitive trigger, "show a friend")
- [ ] Does every customer interaction leave a trace that attracts the next customer?
- [ ] What is the viral coefficient? (Each user creates how many new users?)

## 2. FLYWHEEL EFFECTS
- [ ] Does this feature compound over time? (More users = more data = better product = more users)
- [ ] Does it create lock-in through investment? (Data uploaded, customizations made, history built)
- [ ] Does it reduce CAC over time? (Word of mouth, organic discovery, badge impressions)
- [ ] Does it increase LTV? (Higher engagement, lower churn, expansion revenue)

## 3. ENTRY POINTS
- [ ] How many ways can someone discover this? (Search, referral, competitor trigger, badge, market report, AI citation, QR code, social share)
- [ ] For each entry point, what's the emotional state and what content matches it?
- [ ] Are there programmatic entry points that generate themselves? (Micropages, market reports)
- [ ] Is every entry point AEO-optimized? (Structured data, question-answer format, cited sources)

## 4. RETENTION HOOKS
- [ ] What brings them back daily/weekly? (Monday email, score updates, streak)
- [ ] What creates anticipation? (Sunday text, "your brief is generating")
- [ ] What creates loss aversion for leaving? (History, data, relationships, streak)
- [ ] What creates identity-level belonging? (Foundation, community, "I'm an Alloro business")

## 5. REVENUE MECHANICS
- [ ] Is pricing matched to value by vertical? ($2K for specialists, lower for others?)
- [ ] Is there expansion revenue? (More locations, add-ons, integrations)
- [ ] Is there channel revenue? (Partners, agencies, certified consultants)
- [ ] Does $50/subscription to Foundation create mission-level loyalty?

## 6. COMPETITIVE MOAT
- [ ] What's impossible to copy? (Data accumulation, community, agent intelligence, trust)
- [ ] What gets stronger with time? (Review themes, market reports, heuristic accuracy)
- [ ] What has network effects? (More businesses in a market = better competitive intelligence for all)
- [ ] Does this create the category or compete in one? (Business Clarity, not business intelligence)

## 7. UNDENIABLE MOMENTS
- [ ] Where's the Oz moment? (The thing they didn't know about their own business)
- [ ] Where's the relief moment? (Someone is watching while they sleep)
- [ ] Where's the share trigger? (I can't NOT tell someone about this)
- [ ] Where's the identity moment? ("If you have a business, you're an entrepreneur")

## 8. SELF-OPTIMIZATION
- [ ] Can this feature measure its own effectiveness?
- [ ] Can the Dream Team agents improve it automatically?
- [ ] Is the 1% daily improvement built into the architecture?
- [ ] What does the Kenji test loop look like? (Hypothesis, test, measure, learn)

## 9. CROSS-REFERENCE
- [ ] Checked against Buyer Psychology Architecture (7 laws)?
- [ ] Checked against UX/UI Design Standards?
- [ ] Run ICP Simulation Engine (7 personas)?
- [ ] Cross-referenced with Notion HQ existing specs?
- [ ] Reviewed competitive landscape (Owner.com, ServiceTitan, Zapier)?

## 10. FIRST PRINCIPLES
- [ ] Does this serve the mission? (Give every business owner the life they set out to build)
- [ ] Does this create Business Clarity or just add features?
- [ ] Would the ICP at 10pm on Sunday feel relief or confusion?
- [ ] Is this Google or is this the library?

---

If all 10 sections are checked: write the spec.
If any section has gaps: research and fill before writing.
The "one more thing" pattern stops here.
